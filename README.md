# Review Tag | Backend

Review Tag는 영화를 비롯한 다양한 콘텐츠를 손쉽게 탐색하고, 직접 남긴 리뷰와 별점, 가격 평가를 기록할 수 있을 뿐 아니라, 퀴즈와 포인트 적립까지 경험할 수 있도록 기획한 웹 플랫폼입니다. 본 프로젝트는 KH정보교육원에서 5명이 참여한 파이널 팀 프로젝트로 진행되었으며, 지금 소개드리는 이 저장소는 Review Tag의 백엔드 구현 내용을 정리했습니다

## 프로젝트 소개

- 개발 기간: 2025.11.10 ~ 2025.12.22  
- KH정보교육원 5인 파이널 프로젝트  
- TMDB API를 활용한 콘텐츠 탐색부터 리뷰 작성, 퀴즈, 포인트 시스템까지 하나의 서비스에서 이용할 수 있도록 설계한 웹 플랫폼입니다.  
- 사용자가 리뷰만 남기는 데서 그치지 않고, 포인트 적립과 활용, 각종 랭킹 참여 등으로 자연스럽게 서비스 내 활동을 확장할 수 있도록 흐름을 구성했습니다.

## 전체 구현 기능

- TMDB API를 기반으로 한 콘텐츠 검색, 장르별 탐색 및 상세 정보 제공  
- 리뷰 작성, 수정, 삭제, 별점 및 가격 평가, 좋아요 기능  
- 게시글과 댓글의 CRUD, 신고 처리 등 커뮤니티 기능  
- 사용자별 퀴즈 생성, 풀이, 기록 저장, 퀴즈 신고 및 랭킹 시스템  
- 회원가입, 로그인, JWT 인증, 마이페이지와 프로필, 개인 활동 내역 조회  
- 관심 콘텐츠 저장(찜), 리뷰·퀴즈·포인트 등 랭킹 기능  
- 출석체크, 일일 퀘스트, 룰렛, 포인트 선물, 상점, 위시리스트, 인벤토리, 아이콘, 포인트 적립·차감 이력 등 다양한 포인트 시스템  
- 관리자 전용 기능(회원 관리, 글·리뷰·퀴즈 신고 관리, 포인트 지급·회수, 상품 및 인벤토리 관리)

---

## 프로젝트 소개

- 영화 및 콘텐츠 리뷰와 가격 평가를 한데 모은 올인원 서비스입니다.
- 사용자 개개인의 활동이 포인트 적립과 소비로 자연스럽게 연결되도록 구조를 설계했습니다.
- 로그인 사용자는 출석, 일일 퀘스트, 상점 구매와 인벤토리, 아이콘 수집, 이력 조회, 랭킹 등 다양한 기능을 이용할 수 있습니다.

## 구현에서 중점을 둔 점

- 프로젝트 전체에서 인증 로직을 통일하고, 포인트 API가 동일한 방식으로 작동하도록 아키텍처를 설계했습니다.
- 포인트 증감과 이력 저장이 어떤 화면에서든 같은 규칙으로 처리되도록 세부 로직을 맞췄습니다. 덕분에 서비스의 신뢰도를 높일 수 있었습니다.
- 구매, 환불, 아이템 사용, 관리자 지급 등 여러 케이스도 모두 하나의 포인트 규칙 아래에서 처리되도록 일원화했습니다.
- 일일 퀘스트와 보상 수령 기록 역시 하루 단위로 중복 없이 정확하게 처리하여 사용자가 불편함 없이 이용할 수 있게 했습니다.
