# Review Tag | Backend

Review Tag는 영화와 다양한 콘텐츠에 대한 리뷰를 남기고, 별점과 가격 평가, 퀴즈, 포인트까지 한 번에 경험할 수 있는 리뷰 플랫폼입니다. 이 서비스에서 사용자가 남긴 모든 활동과 평가가 자연스럽게 하나의 흐름으로 이어지도록 팀 프로젝트를 진행했습니다.

저는 백엔드 개발자로서, 이 플랫폼의 포인트 시스템 전반을 설계하고 구축하는 역할을 맡았습니다. 사용자가 출석을 하거나 퀘스트를 완료해 얻은 포인트가 상점, 인벤토리, 이력 등 여러 기능에서 일관된 기준으로 작동하도록 만드는 것이 가장 큰 목표였습니다. 한 번 쌓인 포인트가 서비스를 이용하는 내내 같은 규칙 아래에서 적립, 소비, 이력 관리가 이뤄지도록 세심하게 설계했습니다.

---

## 프로젝트 소개

- 영화 및 콘텐츠 리뷰와 가격 평가를 한데 모은 올인원 서비스입니다.
- 사용자 개개인의 활동이 포인트 적립과 소비로 자연스럽게 연결되도록 구조를 설계했습니다.
- 로그인 사용자는 출석, 일일 퀘스트, 상점 구매와 인벤토리, 아이콘 수집, 이력 조회, 랭킹 등 다양한 기능을 이용할 수 있습니다.

---

## 담당 역할

- 출석 체크 및 일일 퀘스트 포인트 적립 로직 직접 구현
- 상점 구매, 아이템 사용/환불/선물 기능을 위한 API 개발
- 포인트 잔액과 이력 관리를 모든 기능에서 공통적으로 처리
- 아이콘 뽑기 및 장착 흐름 연동
- 관리자 페이지에서 포인트 조정, 상품 관리, 자산 조회 기능 개발

---

## 사용 기술

- Spring Boot
- MyBatis
- Oracle
- JWT
- MVC 패턴
- Git, GitHub

---

## 구현에서 중점을 둔 점

- 프로젝트 전체에서 인증 로직을 통일하고, 포인트 API가 동일한 방식으로 작동하도록 아키텍처를 설계했습니다.
- 포인트 증감과 이력 저장이 어떤 화면에서든 같은 규칙으로 처리되도록 세부 로직을 맞췄습니다. 덕분에 서비스의 신뢰도를 높일 수 있었습니다.
- 구매, 환불, 아이템 사용, 관리자 지급 등 여러 케이스도 모두 하나의 포인트 규칙 아래에서 처리되도록 일원화했습니다.
- 일일 퀘스트와 보상 수령 기록 역시 하루 단위로 중복 없이 정확하게 처리하여 사용자가 불편함 없이 이용할 수 있게 했습니다.
